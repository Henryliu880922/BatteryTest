# 續航力測試工具 PWA 完整版

功能：
- 自動或手動記錄電量
- 測試計時
- 已消耗電量
- 平均每小時耗電速度
- 預估總續航
- 預估剩餘續航
- 耗電趨勢圖
- 測試紀錄
- CSV 匯出
- localStorage 自動保存
- PWA 離線使用
- 可加入 iPhone / Android 主畫面

GitHub Pages：
1. 將所有檔案上傳到 repository 根目錄。
2. Settings → Pages。
3. Deploy from a branch。
4. main / root。
5. 用 Safari 或 Chrome 開啟 GitHub Pages 網址。
6. iPhone：分享 → 加入主畫面。
