# 續航力測試網站

這是一個可在手機與電腦瀏覽器使用的續航力測試工具。

## GitHub Pages 發布

1. 建立新的 GitHub repository。
2. 上傳本專案內的 `index.html`。
3. 進入 **Settings → Pages**。
4. 在 **Build and deployment** 選擇 **Deploy from a branch**。
5. Branch 選擇 `main`，資料夾選擇 `/ (root)`。
6. 按下 **Save**。

發布後網址通常為：

`https://你的GitHub帳號.github.io/repository名稱/`

> iPhone Safari 無法讓網頁直接讀取系統電量，因此會使用手動輸入模式。
